# Privacy Policy for Xtag

Xtag displays and edits user-controlled trust labels for X/Twitter accounts.

The extension reads visible X/Twitter account handles on pages you visit so it can display local trust badges.

Labels created in the extension, including source metadata for imported labels, are stored in Chrome local extension storage (`chrome.storage.local`) on your device.

If you explicitly publish a share link, your selected local label snapshot may be uploaded to the configured HTTPS share backend and returned as a public link. Anyone with that link may view the shared label snapshot. Importing someone else's snapshot opens a preview in the extension and does not merge labels until you confirm.

The extension does not sell user data. The extension does not load or execute remotely hosted code. The extension does not use Google APIs. If Google API data is ever introduced in the future, the use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.
